How to solve famous computer science algorithms by javascript?

There are JavaScript implementations of different famous computer science algorithms.

Algoritmes divided by Data structures: Array, Number, String, ...

Tests provided by Jest.

List of questions:

  • Array

    1. Merge two sorted arrays + test
    2. Find an index in array

  • Number

    1. Factorialize a Number: with recursion, a while loop, a for loop + test.
    2. Translate a number by alphabetic code to a string, for example, solutions(123) --> 'abc', 'mc', 'ax' + test.

  • String

    1. Count the number of occurrences of all characters in a string.
    2. Check if is given string is a palindrome with ignoring spaces, e.g. 'race car' would be palindrome.
    3. Reverse a string: with a decrementing for loop and recursion + test.

Algorithms rules:

The definitive javascript algorithmes for your next developer interview.

I propose several solutions and explain step-by-step what happens under the hood.

The idea don't use the default, in-built methods of Javascript, for example, sort(), reverse() and etc.

Number

4
  1. Question: How would you verify a prime number?
    Prime number
    is a natural number greater than 1 and only divisible by itself
    Solutions:
    • Prime number with a while loop runtime O(n).
    • Prime number with a for loop runtime O(n), O(sqrt(n)).
  2. Question: Three ways to Factorialize a Number in js: return the factorial of the provided integer.
    Factorial
    is product of every whole number from 1 to n. For example, if n is 3 then 3! is 3 x 2 x 1 = 6
    Recursion
    is an important programming technique, in which a function calls itself.
    Solutions:
    • factorialize a number with recursion
    • factorialize a number with a while loop
    • factorialize a mumber with a for loop
  3. Question: Print all positive solutions to the equation a^3+b^3 = c^3 + d^3, where a, b, c, d lie between [0, 10^5].
  4. Question: Translate a number by alphabetic code to a string, for example, solutions(123) --> 'abc', 'mc', 'ax'.
